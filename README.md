# Unity RPG Prototype

使用 Unity 2022 開發的戰棋 RPG 原型。

## 目前功能

- 程序式格子地圖生成系統
- 地板點擊選取機制
- 單位佔位判定
- 角色移動系統
- 角色選取系統（點擊角色選取/取消選取）
- 移動範圍顯示（曼哈頓距離菱形範圍）
- 回合制系統（玩家/敵人回合切換）
- 敵人AI（每回合自動追蹤玩家）
- 中文UI顯示（TextMeshPro）
- 玩家移動範圍顯示（藍色）
- 點擊敵人顯示威脅範圍（紅色）
- 點擊冷卻機制（防止連點bug）
- 回合限制（玩家回合無法操控敵人）
- 戰鬥系統（攻擊、受傷、死亡）
- HP與傷害計算
- 勝負判定（消滅所有敵人獲勝）
